# trydjango110

A tutorial of django 1.1x
一个Django入门教程，基于Django1.1x实现一个简单的BBS项目。

## 目录结构
    trydjango110
    │
    ├── doc  # 视频文档
    ├── pics  # 视频截图文件
    ├── static  # 项目静态文件目录
    ├── static_assets  # 项目的总静态文件
    ├── templates  # 项目模板文件
    ├── bbs # django project
    │   ├── __init__.py
    │   ├── settings.py  # 主配置文件
    │   ├── urls.py  # 主路由文件
    │   └── wsgi.py
    ├── db.sqlite3  # 数据库文件
    ├── requirements.txt  # 项目依赖模块清单
    └── manage.py  # 管理文件
